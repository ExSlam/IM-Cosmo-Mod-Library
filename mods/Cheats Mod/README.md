# Cheats Mod

Adds cheat actions to the Mod Buttons action hub.

## Requires

- Mod Buttons

## Actions

- Add 1B yen
- Add 10k fans
- Add 10k fame
- Add 100 buzz
- Reset special event cooldowns
- Set active idol physical stamina to 100
- Set active idol mental stamina to 100
- Reduce general scandal points by 1
- Add 1 general scandal point
- Add 1 scandal point to all idols
- Set all scandal points to 0
- Increase active idol influence by one level
- Increase active idol friendship by one level
- Increase active idol romance by one level
- Add 25k EXP to all staff skills
- Add 1k points to every research type
