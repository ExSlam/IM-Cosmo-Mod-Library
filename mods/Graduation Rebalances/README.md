# Graduation Rebalances

`Graduation Rebalances` pushes graduation dates back for high-performing idols.

## Player-facing behavior

- Reworks graduation scheduling around idol performance and long-term potential.
- Extends graduation windows for stronger idols instead of letting vanilla timing resolve too aggressively.

## Build

Project file:
- `mods/Graduation Rebalances/Graduation Rebalances.csproj`

Example command:
- `dotnet build "mods/Graduation Rebalances/Graduation Rebalances.csproj" -c Release`
