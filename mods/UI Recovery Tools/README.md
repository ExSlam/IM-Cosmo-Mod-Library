# UI Recovery Tools

`UI Recovery Tools` adds configurable recovery hotkeys and an error overlay for stuck UI states.

## Player-facing behavior

- Adds single-key recovery actions for common soft-lock style UI failures.
- Exposes runtime feedback through an overlay so the current recovery state is visible in-game.

## Build

Project file:
- `mods/UI Recovery Tools/UI Recovery Tools.csproj`

Example command:
- `dotnet build "mods/UI Recovery Tools/UI Recovery Tools.csproj" -c Release`
